# Airline Operations & Revenue Analytics (SQL Project)

## Project Overview
This project simulates an airline database designed to analyze:

- Flight punctuality (OTP)
- Average delays
- Revenue by route
- Customer segmentation
- Booking performance
- Cancellation rates

## Database Design
The project includes:

- Airports
- Routes
- Aircraft
- Customers
- Flights
- Bookings
- Tickets

## SQL Skills Applied
- SELECT / DISTINCT
- WHERE / AND / OR / NOT
- INSERT / UPDATE / DELETE
- JOINs (INNER, LEFT, RIGHT)
- GROUP BY / HAVING
- Aggregate functions
- Subqueries (EXISTS, ANY, ALL)
- UNION / UNION ALL

## Future Expansion
This dataset will be used to build:
- Power BI dashboards
- Revenue analysis
- Operational KPIs
